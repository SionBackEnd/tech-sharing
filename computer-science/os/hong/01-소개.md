## 강의소개
- 운영체제란 컴퓨터 하드웨어 바로 위에 설치되는 소프트웨어 계층으로 모든 컴퓨터 시스템의 필수적인 부분이다.
- 운영체제의 목적은 컴퓨터 시스템의 자원을 효율적으로 관리하는 것이다.
- 본 강의 목표는 운영체제의 개념과 역할, 운영체제를 구성하는 요소와 그 알고리즘의 핵심을 학습하는 것이다.

## 운영체제 소개

### 운영체제 분류
- 동시 작업 가능 여부
  - 단일 작업
  - 다중 작업
- 사용자 수
  - 단일 사용
  - 다중 사용
- 처리 방식
  - 일괄 처리
  - 시분할 : 우리의 운영체제
  - 실시간 : 특수목적

### 운영체제 예
- UNIX
  - 대형 컴퓨터 용도로 만들어져서 다중 작업, 다중 사용 가능
  - 대부분이 C언어로 만들어짐
  - 오픈소스
  - 높은 이식성
  - Linux
- DOS
- MS Windows

### 운영체제의 구조
- CPU 스케줄링
- 메모리 관리
- 파일 관리
- 입출력 관리 (인터럽트)
- 프로세스
